### Homework 5

Створіть простий сценарій перевірки доступу до фільму з віковим обмеженням 18+.

Через prompt запитайте вік користувача.
Якщо користувач натиснув Cancel або нічого не ввів → показати alert("Вік не вказано") і на цьому завершити.
Якщо вік менше 18:
Запитайте через confirm: "Вам менше 18. Чи є з вами дорослий, який дозволяє перегляд?"
Якщо користувач натиснув OK → alert("Доступ дозволено з дозволу дорослого.")
Якщо Cancel → alert("Доступ заборонено.")
Якщо вік 18 або більше → alert("Доступ дозволено. Приємного перегляду!")
