Через prompt запитує в користувача його вік.
Конвертує введене значення в число (використати Number або унарний +).
Якщо користувач ввів не число (NaN, порожній рядок, скасував ввід) — вивести через alert: "Некоректний вік".
Якщо все ок — вивести повідомлення: "Через 5 років вам буде: X", де X – вік + 5.
Обовʼязково: – Використати Number(...) або унарний + для конвертації. – Перевірити isNaN(...).
